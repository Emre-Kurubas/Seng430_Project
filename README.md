## 💡 Overview

This system empowers healthcare professionals and students to seamlessly:

- Upload datasets  
- Preprocess healthcare data  
- Train robust ML models  
- Evaluate performance metrics  
- Complete a full ML workflow interactively  

The platform supports **20 distinct clinical domains** and provides an intuitive end-to-end Machine Learning training experience.

---

## 🚀 Tech Stack

### 🌐 Frontend
- **React** + **Vite** (Fast, modern UI)

### ⚙️ Backend
- **FastAPI** (High-performance Python API)

### 🧠 Machine Learning
- **Scikit-learn**
- **Pandas**
- **NumPy**

---

## 🧬 Supported ML Models

We implement 6 core machine learning algorithms:

- 🟢 **K-Nearest Neighbors (KNN)**
- 🔵 **Support Vector Machines (SVM)**
- 🟡 **Decision Tree**
- 🟠 **Random Forest**
- 🔴 **Logistic Regression**
- 🟣 **Naive Bayes**

---

## 🎯 Project Goals

- ✅ Implement 6 core ML algorithms  
- 🏥 Support **20 clinical domains** with tailored configurations  
- 🔄 Provide a complete **7-step ML workflow**  
- 🎓 Generate a **downloadable certificate** upon successful module completion  
- 🧩 Build an accessible ML training tool tailored for the healthcare sector
